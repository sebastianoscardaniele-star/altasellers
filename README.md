# Seller Master Lite

Aplicacion Vite + React liviana para administrar marcas, sellers, master seller y prestaciones editables.

## Uso local
npm install
npm run dev

## Build
npm run build

## Deploy en Vercel
Subir este proyecto a GitHub e importar el repositorio en Vercel. Framework: Vite. Build command: npm run build. Output: dist.

## Datos
La app guarda cambios en localStorage. Permite exportar e importar JSON para respaldo o migracion.

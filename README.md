# Altas Tiendas - Seller Master

App Vite + React basada en `Consolidado Altas Tiendas(1).xlsx`.

## Uso
```bash
npm install
npm run dev
```

## Deploy en Vercel
Subir a GitHub e importar el repo en Vercel. Build command: `npm run build`. Output: `dist`.

## Funciones
- Solapas por marca/hoja del Excel.
- Selector de seller con un click.
- Edición de todos los campos.
- Búsqueda global y por solapa.
- Apartado Master Seller basado en hoja Maestro-Seller.
- Exportar/Importar JSON.
- Importar Excel actualizado.
- Persistencia local en navegador.
